### 🛠 Tech Stack

- **Styling**: CSS Modules
- **Tooling**: ESLint, Prettier

### 🧪 Getting Started

```bash
git clone https://github.com/w-gluza/portfolio
cd portfolio
npm install
npm run dev
```
